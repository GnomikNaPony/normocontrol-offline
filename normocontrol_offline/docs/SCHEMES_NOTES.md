# Схемы проекта

Изучены фотографии `2026-06-16 01.00.32.jpg` и `2026-06-16 01.00.38.jpg`.

## Зафиксированные требования

- Автоматический нормоконтроль документов.
- При изменении исходных СТО и РУ автоматически находить затронутые документы,
  готовить изменения и показывать их человеку перед применением.
- Хранить векторную базу документов и базу исправлений.
- Использовать локальную модель, которая уверенно работает с русским языком.
- Не выполнять fine-tuning на первом этапе; использовать RAG, граф знаний и
  проверяемые правила.
- Рассмотреть специализированных агентов с общей базой знаний.
- Интерфейс должен быть простым, понятным и не перегруженным.

## Принятое решение

Точные массовые изменения выполняет детерминированное ядро. Локальная LLM
объясняет и предлагает содержательные исправления, но не меняет документы без
подтверждения человека.
