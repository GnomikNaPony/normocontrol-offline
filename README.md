# Нормоконтроль Offline

Репозиторий содержит прототип локальной программы нормоконтроля документов и
материалы для обучения/проверки.

Локальная папка проекта переименована для удобного доступа через SSH:

```bash
cd ~/Desktop/normocontrol-offline/normocontrol_offline
./start.sh
```

На macOS можно запускать и двойным кликом по файлу
`normocontrol_offline/start.command`.

## Что теперь умеет программа

- Работает локально в собственном desktop-окне, без браузера и без интернета.
- Создает и открывает SQLite-базы документов.
- Импортирует DOCX, ODT, PDF, TXT/MD, изображения и старые DOC.
- Делает OCR сканов PDF/JPG/PNG/TIFF через локальный Tesseract.
- Извлекает текст, ссылки на ГОСТ/РУ/СТО/НП и разметку Word.
- Ищет устаревшие ссылки и выпускает исправленные DOCX-копии.
- Перед любым применением изменений показывает окно проверки и ждет решения человека.
- Создает отчет `report_*.md` со ссылками на исходники, копии и таблицей замен.
- Обучается на парах “до/после”, но не копит тяжелый текст примеров в базе.
- Проверяет пробелы, пунктуацию, ручную нумерацию, поля Word, колонтитулы,
  таблицы, подписи, стили и OCR-риск.
- Хранит исходные документы как обычные файлы и не перезаписывает их.

## Как взаимодействовать

В основном окне оставлены только четыре кнопки:

- `Добавить` - загрузить документы, стандарты, примеры или схемы.
- `Проверить` - запустить анализ базы и обновить список замечаний.
- `Исправить` - посмотреть план замен, подтвердить его и выпустить копии.
- `Еще` - выбрать базу, обучить на примерах или обновить ссылку на стандарт.

Рекомендуемый сценарий:

1. `Еще` -> `Выбрать или создать базу`.
2. `Добавить` -> загрузить стандарты как `Стандарты`.
3. `Добавить` -> загрузить рабочие документы как `Документы`.
4. `Добавить` -> загрузить пары “до/после” как `Примеры`.
5. `Еще` -> `Обучить на примерах`.
6. `Проверить`.
7. При новой редакции стандарта: `Еще` -> `Обновить ссылку на стандарт`.
8. `Исправить` -> проверить план -> `Подтвердить выпуск копий`.

Быстрый запуск:

```bash
cd normocontrol_offline
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
./start.sh
```
